## 五、估值结果

### 5.1 DCF估值计算

#### 基准情景（WACC=8.5%，永续增长率g=3.0%）

| 年度 | FCF(亿元) | 折现系数 | 现值(亿元) |
|------|----------|---------|----------|
| 2026年 | 22.00 | 0.9217 | 20.28 |
| 2027年 | 27.00 | 0.8496 | 22.94 |
| 2028年 | 32.00 | 0.7832 | 25.06 |
| 2029年 | 37.00 | 0.7219 | 26.71 |
| 2030年 | 42.00 | 0.6655 | 27.95 |
| **第一阶段现值合计** | | | **122.94** |

**终值计算：**
- 2030年FCF = 42.00亿元
- 终值 = 42.00 × (1+3.0%) / (8.5% - 3.0%) = 780.00亿元
- 终值现值 = 780.00 / (1.085)^5 = 519.09亿元

**企业价值 = 第一阶段现值 + 终值现值 = 122.94 + 519.09 = 642.03亿元**

**股权价值 = 企业价值 - 净负债 = 642.03 - 0 ≈ 642亿元**

**每股价值 = 642亿 / 6.64亿股 = 96.68元**

#### 乐观情景（WACC=8.0%，永续增长率g=3.5%）

| 年度 | FCF(亿元) | 折现系数 | 现值(亿元) |
|------|----------|---------|----------|
| 2026年 | 22.00 | 0.9259 | 20.37 |
| 2027年 | 27.00 | 0.8573 | 23.15 |
| 2028年 | 32.00 | 0.7938 | 25.40 |
| 2029年 | 37.00 | 0.7350 | 27.20 |
| 2030年 | 42.00 | 0.6806 | 28.59 |
| **第一阶段现值合计** | | | **124.71** |

**终值计算：**
- 终值 = 42.00 × (1+3.5%) / (8.0% - 3.5%) = 987.00亿元
- 终值现值 = 987.00 / (1.08)^5 = 671.15亿元

**企业价值 = 124.71 + 671.15 = 795.86亿元**

**每股价值 = 795.86 / 6.64 = 119.86元**

#### 保守情景（WACC=9.0%，永续增长率g=2.5%）

| 年度 | FCF(亿元) | 折现系数 | 现值(亿元) |
|------|----------|---------|----------|
| 2026年 | 22.00 | 0.9174 | 20.18 |
| 2027年 | 27.00 | 0.8417 | 22.73 |
| 2028年 | 32.00 | 0.7722 | 24.71 |
| 2029年 | 37.00 | 0.7084 | 26.21 |
| 2030年 | 42.00 | 0.6499 | 27.30 |
| **第一阶段现值合计** | | | **121.13** |

**终值计算：**
- 终值 = 42.00 × (1+2.5%) / (9.0% - 2.5%) = 650.77亿元
- 终值现值 = 650.77 / (1.09)^5 = 422.80亿元

**企业价值 = 121.13 + 422.80 = 543.93亿元**

**每股价值 = 543.93 / 6.64 = 81.92元**

## 八、投资建议

1. **存储行业周期波动风险**：存储业务占营收超70%，行业周期性影响显著
2. **高估值风险**：当前股价对应2026年PE约61倍，处于历史高位
3. **技术竞争风险**：AI芯片、高端MCU等领域竞争激烈
4. **供应链风险**：Fabless模式依赖外部代工厂
5. **并购整合风险**：苏州赛芯整合效果待观察
